1. События в JavaScript:
Когда кликаешь мышкой по кнопке на сайте, этот клик проходит два пути:
cначала сверху вниз, от окна браузера -> к кнопке (это фаза захвата), потом снизу вверх: от кнопки -> обратно к окну (это фаза всплытия)
Большинство кодов срабатывают при всплытии. Это позволяет вешать один обработчик на родителя для многих кнопок. Останавливать событие, чтобы оно не шло дальше перехватывать события раньше других

2. Promise и асинхронный код
Promise - это обещание результата.
Состояния Promise:
Ожидание
Выполнено
Ошибка 

Event Loop - это диспетчер, который решает, какую задачу выполнять следующей. Он постоянно проверяет: есть ли что сделать? Благодаря ему браузер не зависает при загрузке.

3. ООП
ООП - это способ писать программы как набор объектов:
Свойства
Методы
принципы ООП:
Инкапсуляция - прячем сложное внутри, наружу показываем только нужное
Наследование - дети получают свойства родителей
Полиморфизм - одно действие, разные результаты
Абстракция - работаем с понятиями, а не с деталями

В JavaScript ООП работает через прототипы - каждый объект может ссылаться на другой и брать у него свойства.

4. Как браузер открывает сайт

Когда вводишь адрес сайта:
Браузер спрашивает DNS-сервер "где этот сайт?"
Получает IP-адрес (как адрес дома)
Отправляет запрос на сервер
Сервер присылает HTML, CSS, JS
Браузер рисует страницу из этих частей
Загружает картинки, видео
Чтобы ускорить:
Кэширование (сохранять части сайта на компьютере)
CDN (хранить копии сайта ближе к пользователю)
Сжаие файлов
Проблемы безопасности:
Фишинг (поддельные сайты)
XSS (вредоносный код на странице)
CORS (ограничение доступа между разными сайтами)
Кража cookies (воровство сессий)
HTTPS шифрует связь между браузером и сервером, чтобы никто не мог подслушать.
